# Movie-Recommendation-System
This project is a content-based Movie Recommendation System that suggests similar movies based on a selected title. It includes EDA, text vectorization, and cosine similarity to generate accurate recommendations, along with a simple and interactive Streamlit interface for users.

## Project Workflow
- Data preprocessing
- Text vectorization
- Similarity calculation (using cosine similarity)
- Interactive UI built with Streamlit

## Tech Stack
- Python
- NumPy, Pandas
- Scikit-learn
- Streamlit
